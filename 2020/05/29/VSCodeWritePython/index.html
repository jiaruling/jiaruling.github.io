用 VSCode 写 Python 代码

发表于 2020-05-29 分类于 VSCodeWriteCode

VSCode 宇宙第一编辑器

  • VSCode的功能非常的强大，丰富的插件几乎可以满足我们所有的开发需要，最重要的它还是免费的。关于VScode的介绍就不多说了，在这里主要讲解VScode中编写和运行python程序，以及一些关于python程序的扩展。

    python的常用的开发工具介绍

  • 如果用python做项目级别的开发建议使用 【项目的管理和开发都是超级非常棒的，如果非要有什么不足的话那就是软件比较大，占用更多的内存空间】；

  • 如果用python做数据科学方面的研究建议使用 【很好的 和 工具。我自己现在的电脑上都不安装原生的python解释器了，直接使用Anconda来帮助我管理多个python虚拟环境】 当中的 或者 【二者的差别目前还没有深入研究，感觉是 要更加高级一些。二者都可以在编写代码的同时显示代码运行的结果，同时支持生成多种文档，便于分享自己编写的代码和代码运行的结果。不足之处在于编写代码的时候，代码的自动补全功能没有 那么强大】；

  • 如果用python做一些小的测试程序【只需要一个.py文件就够了，也就几十行代码】，那么完全没有必要使用 这么强大的IDE开发工具。 是轻量级的编辑器，占用内存小，打开速度比 更加快；可定制化程度高，开发者可以根据自身的开发需求去安装自己的需要的插件；插件种类丰富【只有想不到，没有找不到】，完全能够满足开发者的需求。
