数据库

数据库介绍

  1. 什么是数据库系统？

    数据库系统(DBMS)是指能为用户提供 的系统。

  2. 什么是关系型数据库系统？

    关系型数据库系统(RDBMS)指使用了 的数据库系统。关系模型中，数据是分类存放的，数据之间可以有联系。关系型数据库系统可以有效组织和管理大量复杂的数据，所以关系型数据库才是最重要的数据库产品。

  3. 主流关系型数据库。
    • DB2
    • Oracle
    • MySQL
    • SQL server
  4. 什么是NoSQL数据库系统？

    NoSQL数据库是指数据分类存放，但是数据之间没有关联关系的数据库系统。

  5. 主流的NoSQL数据库
    • Redis
    • MemCache
    • MongoDB
    • Neo4J
  6. 什么是SQL语言？

    SQL语言是用于 和 数据的标准的计算机语言。

  7. SQL语言的分类
    • DML
      • 添加
      • 修改
      • 删除
      • 查询
    • DCL
      • 用户
      • 权限
      • 事务
    • DDL
      • 逻辑库
      • 数据表
      • 试图
      • 索引

MySQL的时区设置

DataGrip 是一款非常好用的工具，但是经常在连接数据库的时候对于数据库的时区有要求。MySQL 默认的时区是UTC时区，但是我们国家时采用的 东八区 的时间。所以要在 DataGrip 中使用MySQL必须要对MySQL进行时区的设置，要不然 DataGrip 无法连接到MySQL数据库。

  • 查看MySQL时区

  • 修改MySQL的时区

    1. 永久修改: 修改mysql的配置文件my-default.ini，添加：default-time-zone=’+08:00’，重启mysql生效,注意一定要在 [mysqld] 之下加 ，否则会出现 unknown variable ‘default-time-zone=+8:00’

      重启mysql

    2. 临时的修改: 执行mysql命令 set global time_zone=’+08:00’，立即生效，重启mysql后失效
