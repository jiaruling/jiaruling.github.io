如何写博客

程序员为什么要写博客?

  • 原因很简单:

    1. 计算机的知识体系过于庞大，对于一个普通人来说，太多的知识点我们是根据记不住的。
    2. 很多知识点都是以代码的形式展示，我们需要高效的管理我们学习过的代码块。
    3. 编程不是一次性就能够学的很完美，编程的学习是需要我们在不断地迭代中去完善我们的知识体系。
    4. 分享个人经验，建立 ，提升自己的影响力。

  • 为什么要管理写过的代码?

    1. 整理写过的代码，整理过程中可以巩固自己学习的内容，日后方便查找和复习。
    2. 我们在编写代码的过程中需要用到以前写过的代码，直接在自己的博客中找到相应的代码块复制粘贴即可。

工具介绍

  • 博客框架: Hexo 【快速、简洁且高效的博客框架】
  • 博客主题: NexT 【当然还有很多其他的主题可供选择】
  • 编辑工具: VSCode 【宇宙第一编辑器,专业人士使用】
  • 主要语言: Markdown 【轻量级标记语言,语法比HTML简单,非常适合书写代码块】
  • 辅助语言: HTML / CSS 【Markdown语法不能满足特定需求时使用】
  • 菜单图标: icon【菜单栏中用到的小图标只能是这个网站里已存在的图标】
